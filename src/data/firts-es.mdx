## Estás cansado del código CSS desorganizado y desordenado? Si es así, es posible que desees considerar el uso de la metodología BEM para ayudarte a estructurar tu código de una manera más modular y fácil de mantener. Y para facilitar aún más la implementación de BEM, puedes usar el asistente de BEM, una extensión de Visual Studio que proporciona autocompletado para las clases de BEM.

Aquí te mostramos cómo comenzar con el asistente BEM:

### Paso 1: Instala la extensión auxiliar de BEM

Primero, debes instalar la extensión BEM Helper en Visual Studio. Puedes hacerlo yendo al menú "Extensiones", seleccionando "Administrar extensiones" y luego buscando "BEM Helper" en la pestaña Marketplace. Haz clic en "Instalar" y reinicia Visual Studio.

Link: [BEM Helper](https://marketplace.visualstudio.com/items?itemName=Box-Of-Hats.bemhelper)

### Paso 2: Comienza a usar las clases BEM

Una vez que se instala la extensión, puedes comenzar a usar clases BEM en tu código CSS. BEM significa Bloque, Elemento, Modificador y la convención de nomenclatura te ayuda a identificar el propósito de cada clase.

Por ejemplo, supongamos que tienes un bloque llamado "menú" con dos elementos: "elemento" y "enlace". Para diseñar el bloque "menú", usarías la clase ".menu". Para diseñar el elemento "elemento", usarías la clase ".menuitem". Y si quisieras agregar un modificador al elemento "item" para cambiar su apariencia, usarías la clase ".menu\*\*item--active".

![Ejemplo de código BEM](/blog/bem/1677184266953.png)

### Paso 3: Usa BEM Helper para autocompletar

Para facilitar el uso de las clases BEM, puedes usar el asistente BEM para autocompletar. Cuando comiences a escribir una clase BEM en tu código CSS, la extensión mostrará un menú desplegable con sugerencias para las clases BEM que coincidan con lo que estás escribiendo. Luego puedes seleccionar la clase que desees y el ayudante de BEM la completará por ti.

![BEM Helper en acción](/blog/bem/1677184426537.gif)

### Paso 4: Disfruta de un código CSS limpio y organizado

Al usar la metodología BEM y el asistente BEM en Visual Studio, puedes crear un código CSS limpio y organizado que es más fácil de mantener y comprender. Además, puedes ahorrar tiempo utilizando la función de autocompletar para agregar rápidamente clases BEM a tu código.

En conclusión, si estás buscando una manera de mejorar tu organización y productividad de CSS, considera usar la metodología BEM y el asistente BEM en Visual Studio. Con un poco de práctica, pronto estarás escribiendo un mejor código CSS en muy poco tiempo.
